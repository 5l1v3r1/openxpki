Subversion structure

We use the version control system from BerliOS. The actual structure under trunk is the following one:

trunk/perl-modules/
trunk/perl-modules/core/
trunk/perl-modules/core/trunk/
the most up-to-date core development
www.openxpki.org/
www.openxpki.org/trunk/update_www_openxpki_org
A script to update the web servers
www.openxpki.org/trunk/htdocs
The web pages itself
