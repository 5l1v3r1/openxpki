OpenXPKI uses the sourecForge.net platform for various things. The major aspects why we choose SourceForge.net is the stability and professional level which the SourceForge.net staff provides to the projects.

  • OpenXPKI at SourceForge
  • Mailing Lists
  • Bug Tracking

  • ViewCVS (but with Subversion content)
  • WebSVN (sponsored by Cryptocom)

Subversion structure

We use the version control system from SourceForge. The actual structure under trunk is the following one:

trunk/perl-modules/
trunk/perl-modules/core/
trunk/perl-modules/core/trunk/
the most up-to-date core development
www.openxpki.org/
www.openxpki.org/trunk/update_www_openxpki_org
A script to update the web servers
www.openxpki.org/trunk/htdocs
The web pages itself
