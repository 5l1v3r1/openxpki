WWW Mirrors

This website is maintained via the Sourceforge.net svn repository and hosted on several independant www servers to increase availability.
If you wish to mirror these pages you will need to set up an script on your webserver that pulls a copy from the repository.
You can get the script from the svn archives to, see Resources section for details.
If you have established a mirror please send a mail to openxpki-users@lists.sf.net so we can add your server to this list and assign a DNS entry.
